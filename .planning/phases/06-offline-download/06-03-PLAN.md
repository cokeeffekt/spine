    **Part A: Add showDownloadedOnly flag to $store.downloads** (already created in Plan 02):

    In the `Alpine.store('downloads', {...})` definition, verify `showDownloadedOnly: false` property exists. If not, add it inside the store object.

    Add two computed-like methods to $store.downloads:
    ```javascript
    getDownloadedCount() {
      return Object.values(this.states).filter(s => s.status === 'complete').length
    },
    getTotalSizeBytes() {
      let total = 0
      for (const s of Object.values(this.states)) {
        if (s.status === 'complete') total += s.sizeBytes
      }
      return total
    },
    toggleDownloadedFilter() {
      this.showDownloadedOnly = !this.showDownloadedOnly
    }
    ```

    **Part B: Update $store.library.filteredBooks getter** to respect the downloaded filter:

    Replace the existing `get filteredBooks()` in `$store.library` with:
    ```javascript
    get filteredBooks() {
      let books = this.books
      // Apply search filter
      if (this.query) {
        const q = this.query.toLowerCase()
        books = books.filter(b =>
          b.title.toLowerCase().includes(q) ||
          (b.author && b.author.toLowerCase().includes(q))
        )
      }
      // Apply "Downloaded" filter (D-07)
      if (Alpine.store('downloads').showDownloadedOnly) {
        books = books.filter(b => Alpine.store('downloads').isDownloaded(b.id))
      }
      return books
    }
    ```
