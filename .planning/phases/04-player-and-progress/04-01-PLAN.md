**Block 1: `progressDB` IIFE (add BEFORE the `alpine:init` listener in the existing `<script>` block)**

Implement raw IndexedDB wrapper (no `idb` library -- per research recommendation). The `progressDB` object with:
- DB_NAME = `'spine-progress'`, DB_VERSION = 1, STORE = `'progress'`
- `open()` — returns cached `_db` or opens new; `onupgradeneeded` creates the `'progress'` object store
- `key(username, bookId)` — returns `${username}::${bookId}` (per D-13, PROG-04)
- `get(username, bookId)` — reads from IndexedDB, returns record or null
- `save(username, bookId, data)` — puts data at the composite key

**Block 2: `Alpine.store('player', {...})` (add inside the existing `alpine:init` listener, after the `library` store)**

State properties:
- `book: null` — currently loaded book object (from `$store.library.selectedBook`)
- `playing: false`
- `currentTime: 0`
- `duration: 0`
- `speed: 1.0`
- `currentChapterIdx: 0`
- `expanded: false`
- `sleepTimer: null` (setTimeout handle — used in Plan 02)
- `sleepMode: null` ('chapter' | number minutes — used in Plan 02)
- `saveInterval: null` (setInterval handle)

Computed getter:
- `get currentChapter()` — returns `this.book.chapters[this.currentChapterIdx]` or null

Methods:
- `initAudio(el)` — stores audio element reference on `window._spineAudio = el` (NOT as a reactive store property -- Alpine proxies break HTMLMediaElement per research Pitfall 1). Sets up event listeners on el:
  - `timeupdate`: updates `this.currentTime = el.currentTime`, recalculates `this.currentChapterIdx` by iterating chapters backwards (find last chapter where `currentTime >= ch.start_sec`)
  - `pause`: sets `this.playing = false`, clears save interval, calls `this._saveProgress()`
  - `ended`: same as pause handler
  - `loadedmetadata`: sets `this.duration = el.duration`

- `async play(book)` — loads and plays a book:
  1. Set `this.book = book`
  2. Set `window._spineAudio.src = '/api/books/' + book.id + '/audio'`
  3. Read saved progress: `const saved = await progressDB.get(Alpine.store('auth').username, book.id)`
  4. Add `canplay` listener with `{ once: true }`:
     - If saved: `el.currentTime = saved.timestamp; this.currentChapterIdx = saved.chapterIdx`
     - Set `el.playbackRate = saved ? saved.speed : 1.0; this.speed = el.playbackRate`
     - Call `el.play(); this.playing = true; this._startSaveInterval()`
  5. Call `el.load()`

- `togglePlay()` — if playing, call `window._spineAudio.pause()`. Else call `window._spineAudio.play()`, set `this.playing = true`, call `this._startSaveInterval()`

- `skip(seconds)` — `el.currentTime = Math.max(0, Math.min(el.currentTime + seconds, el.duration))` (per D-06: +30 / -30)

- `setSpeed(speed)` — set `this.speed = speed`, `window._spineAudio.playbackRate = speed`, then call `this._saveProgress()` to persist speed preference (per D-07, PLAY-06)

- `jumpToChapter(chapterIdx)` — get chapter from `this.book.chapters[chapterIdx]`, set `window._spineAudio.currentTime = ch.start_sec`, update `this.currentChapterIdx = chapterIdx`. If not playing, start playback. (per D-04, PLAY-04)

- `seek(time)` — set `window._spineAudio.currentTime = time` (for seek bar input)

- `async _saveProgress()` — if no book, return. Call `progressDB.save(Alpine.store('auth').username, this.book.id, { timestamp: this.currentTime, chapterIdx: this.currentChapterIdx, speed: this.speed, updatedAt: Date.now() })` (per D-10, D-11, PROG-01)

- `_startSaveInterval()` — clear existing, set `this.saveInterval = setInterval(() => { if (this.playing) this._saveProgress() }, 15000)` (per D-10: every 15 seconds)

- `_clearSaveInterval()` — clearInterval + null

Use the `formatTime()` function from `player-utils.js` (loaded via script tag above). Use `clampSkip()` from player-utils.js in the `skip(seconds)` method instead of inline Math.max/Math.min. Use `getCurrentChapterIdx()` from player-utils.js in the timeupdate handler. Use `progressKey()` from player-utils.js in `progressDB.key()`.
