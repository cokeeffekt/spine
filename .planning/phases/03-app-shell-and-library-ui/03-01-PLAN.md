**1. Update `src/server.ts`:**

Add `import { serveStatic } from 'hono/bun'` at the top of the file (NOT from `@hono/node-server/serve-static` — wrong import for Bun, per Pitfall 1).

Add these two lines AFTER all existing `app.route()` calls (after `app.route("/api", coverRoutes);`), but BEFORE the `if (process.env["NODE_ENV"] !== "test")` block:

```typescript
// Static files — must come AFTER all API/auth routes (per D-20, Pitfall 2)
app.use('/*', serveStatic({ root: './public' }))
app.get('/*', serveStatic({ path: './public/index.html' }))
```

The first line serves files from `public/`. The second is the SPA fallback — any unmatched GET returns `index.html`.

**2. Create `public/manifest.json`** (per D-15, PWA-01, PWA-03):

```json
{
  "name": "Spine",
  "short_name": "Spine",
  "start_url": "/",
  "display": "standalone",
  "theme_color": "#1a1a2e",
  "background_color": "#16213e",
  "icons": [
    { "src": "/icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

**3. Create `public/sw.js`** (per D-16, PWA-02):

```javascript
importScripts('https://storage.googleapis.com/workbox-cdn/releases/7.4.0/workbox-sw.js')

workbox.precaching.precacheAndRoute([
  { url: '/index.html', revision: '1' },
  { url: '/style.css', revision: '1' },
  { url: '/manifest.json', revision: '1' },
  { url: '/icons/icon-192.png', revision: '1' },
  { url: '/icons/icon-512.png', revision: '1' }
])

workbox.routing.registerRoute(
  ({ url }) => url.pathname.startsWith('/api/') || url.pathname.startsWith('/auth/'),
  new workbox.strategies.NetworkFirst({ cacheName: 'api-cache' })
)
```

Create `public/` and `public/icons/` directories first with `mkdir -p public/icons`.
