<!-- Backend API contract from Plan 01 -->

POST /api/scan (admin-only):
- 200 { ok: true } — scan started
- 409 { error: 'Scan already in progress' } — scan already running
- 403 { error: 'Forbidden' } — non-admin
- 401 { error: 'Unauthorized' } — no session

GET /api/scan/progress (admin-only, SSE):
- Content-Type: text/event-stream
- Events:
  - event: start, data: { type: 'start', total: number }
  - event: file, data: { type: 'file', scanned: number, total: number, current: string }
  - event: done, data: { type: 'done', newBooks: number, updatedBooks: number, missing: number, notEnriched: number }

Existing Alpine stores:
- $store.library.loadBooks() — refreshes library grid
- $store.app.view — current view ('library' | 'player' | 'admin')
- $store.auth.username — current user's username

Existing CSS classes (reusable):
- .admin-container, .admin-heading, .admin-status, .btn-primary

    And add the `startScan()` method in x-data:
    ```javascript
    async startScan() {
      this.scanError = ''
      this.scanSummary = null
      const res = await fetch('/api/scan', { method: 'POST' })
      if (res.status === 409) {
        this.scanError = 'A scan is already in progress.'
        return
      }
      if (!res.ok) {
        this.scanError = 'Failed to start scan.'
        return
      }
      this.scanning = true
      this.scanScanned = 0
      this.scanTotal = 0
      this.scanCurrent = ''

      const es = new EventSource('/api/scan/progress')
      this._scanEs = es

      es.addEventListener('start', (e) => {
        const d = JSON.parse(e.data)
        this.scanTotal = d.total
        this.scanScanned = 0
      })
      es.addEventListener('file', (e) => {
        const d = JSON.parse(e.data)
        this.scanScanned = d.scanned
        this.scanTotal = d.total
        this.scanCurrent = d.current
      })
      es.addEventListener('done', (e) => {
        const d = JSON.parse(e.data)
        this.scanning = false
        this.scanSummary = d
        this._scanEs = null
        es.close()
        $store.library.loadBooks()
      })
      es.onerror = () => {
        if (this.scanning) {
          this.scanning = false
          this.scanError = 'Connection lost during scan.'
        }
        this._scanEs = null
        es.close()
      }
    },
    ```
